To use the url shortener, you need:
1.Import the sql file test.sql from the project folder 
	you can see in the article by clicking on this link :
	https://vps.ua/wiki/install-linux-vps/databases/mysql/mysql-import-phpmyadmin/


2.Enter your connection parameters to the DB in the config.php file;

3.Run it through the open server or on any other web server, 
	how to install and configure the open server for work you can see in the article by clicking on this link :  
	https://www.ipipe.ru/info/ustanovka-openserver.html
	Save the files downloaded from the gita to the folder of the open server called "domains" 
	from the catalog "OSPanel"
